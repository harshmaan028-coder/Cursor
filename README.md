# Cursor Website Clone

A frontend replica of the Cursor website built using HTML and CSS.  
This project focuses on recreating the layout, typography, and overall visual design as closely as possible.

---

## 🚀 Features

- Pixel-accurate layout inspired by the original Cursor landing page
- Clean and modern UI design
- Proper use of CSS Grid and Flexbox
- Typography and spacing matching real-world design systems
- Responsive structure (if you added it)

---

## 🛠️ Tech Stack

- HTML5
- CSS3

---

## 🎯 Purpose

The goal of this project was to:

- Practice real-world frontend development
- Improve understanding of layout systems (Grid & Flexbox)
- Learn spacing, alignment, and typography
- Build pixel-perfect UI from an existing design


---

## 💡 Learnings

- Handling layout using CSS Grid and Flexbox
- Managing spacing and alignment like production websites
- Understanding font styling (line-height, letter-spacing)
- Fixing overflow and responsiveness issues
